# Курс "Первая фаза" с расширенными темами

**Время обучения:** 8 месяцев

## Описание курса

Курс предназначен для начинающих веб-разработчиков и охватывает все основные аспекты создания и разработки веб-сайтов и веб-приложений. Он начинается с базовых понятий веб-разработки и постепенно переходит к более сложным темам, включая CSS фреймворки, JavaScript и работу с API. Курс включает теоретические занятия, практические задания и проектные работы, чтобы помочь вам развить навыки и создать портфолио.

## Месяц 1: Основы веб-разработки

## Неделя 1-2: Введение в веб-разработку

### День 1-2 (2-4 часа)

**Что изучается:**
- Введение в веб-разработку
- Построение базовой HTML-страницы
- Стилизация страницы с помощью CSS
- Обсуждение компонентов сайта: клиентская и серверная части

**Практика:**
- **Создание простого HTML-документа:** Создать базовый HTML-документ с заголовком, параграфами, списками и изображениями.
- **Стилизация с CSS:** Добавить стили к созданной HTML-странице, изменив цвета, шрифты и отступы.

### День 3-4 (2-4 часа)

**Что изучается:**
- Основы сетей и протоколов
- Определение сети
- Виды сетей: LAN, WAN, Internet
- Как работает интернет
- Протоколы передачи данных: TCP/IP, HTTP, HTTPS
- Как запросы и ответы обрабатываются в сети

**Практика:**
- **Изучение сетевых инструментов:** Использовать инструменты браузера для анализа сетевых запросов (например, вкладка Network в DevTools).
- **Практика с протоколами:** Выполнить простые запросы через командную строку (например, `curl` или `wget`), чтобы увидеть, как работают HTTP/HTTPS запросы.

### День 5-6 (2-4 часа)

**Что изучается:**
- Обзор инструментов разработчика
- Редакторы кода: VS Code, Sublime Text
- Браузеры: Chrome, Firefox
- Консоли разработчика
- Инструменты для отладки HTML и CSS

**Практика:**
- **Настройка редактора кода:** Настроить редактор (VS Code или Sublime Text) и установить полезные расширения для веб-разработки.
- **Использование инструментов разработчика:** Провести отладку HTML и CSS в браузере, используя консоль разработчика.

### День 7 (2-4 часа)

**Что изучается:**
- Дополнение: Знакомство с Git и GitHub
- Основы версионного контроля
- Создание репозитория на GitHub
- Основные команды Git: `git init`, `git add`, `git commit`, `git push`, `git pull`

**Практика:**
- **Создание репозитория:** Создать новый репозиторий на GitHub и загрузить свою HTML-страницу и CSS файл в этот репозиторий.
- **Основные команды Git:** Попрактиковаться в использовании команд Git для отслеживания изменений, коммитов и отправки на GitHub.

**Проектное задание:**
- Создание HTML-страницы с базовой структурой
- Добавление заголовка, параграфов, изображений и ссылок
- Стилизация страницы с помощью CSS (цвета, шрифты, отступы)

**Практика:**
- Разработать и стилизовать HTML-страницу, используя изученные концепции.
- Постепенно улучшать и добавлять новые элементы, чтобы лучше понять, как работают HTML и CSS.
